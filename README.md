# UniBite
Repository for UniBite. 

**UniBite** is a mobile app that allows students and teachers to pre-order food from in campus cafeterias, helping them 
avoid lines and cut waiting times

## Features
- Check store menus
- Make orders
- Prototype interface focused on students

## Tech Stack
- React Native
- JavaScript
